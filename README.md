# mano-machine-emulator

### Usage

##### Developement mode 

`npm install` to install dependencies

`npm install -g nodemon` to install nodemon package

`npm run dev` to run program in dev environment 

##### Production mode

`npm install` to install dependencies

`npm start` to start the program
